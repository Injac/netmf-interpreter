Welcome to the .NET Micro Framework interpreter

This repository contains the source code to the .NET Micro Framework interpreter engine, Base Class Libraries and the native code Platform and Hardware Abstraction Layers (PAL/HAL) v4.4 and later. (For v4.3 and earlier see the .NET Micro Framework CodePlex site )

NOTE: In order to build the source code in the repository you will need additional binary tools not included in the repository. Our goal is to eliminate the need for these tools in the future so they are not included in the repository and therfore won't be dragged around for the rest of the lifetime of the Micro Framework. Full details can be found in the readme.md file in the repository.

Releases

This is where we will post links to officially released binaries for the SDK. At present there aren't any as v4.4 is still under development. However, the v4.3 installers are available from the .NET Micro Framework CodePlex site
